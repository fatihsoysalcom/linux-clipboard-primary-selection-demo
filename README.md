# Linux Clipboard Primary Selection Demo

This Python Tkinter example demonstrates the distinct clipboard mechanisms in Linux (X11): PRIMARY and CLIPBOARD selections. Users can type and select text in the first box, then use buttons to explicitly copy/paste from CLIPBOARD (like Ctrl+C/Ctrl+V) or implicitly paste from PRIMARY (like middle-click after highlighting). It highlights how Linux manages two separate selections simultaneously, a key difference from other operating systems.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Run from your terminal: `python main.py`
3. Interact with the GUI: Type and select text in 'Text 1', then use the buttons to observe the two distinct clipboard behaviors.
